
# Team Task Manager

A full-stack MERN application with:
- JWT Authentication
- Role-based access
- Project Management
- Task Tracking
- MongoDB Database

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- JWT Authentication

## Backend Setup

```bash
cd backend
npm install
```

Create `.env` using `.env.example`

Run backend:

```bash
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Railway Deployment

### Backend
1. Push backend to GitHub
2. Create Railway Project
3. Add environment variables
4. Deploy

### Frontend
Deploy frontend on Railway or Vercel.
